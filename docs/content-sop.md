# 小红书与朋友圈内容生产 SOP

## 1. 小红书内容智能体

输入：
- 主题：例如保研简历、保研经验、导师套磁。
- 目标人群：保研er、本科生、研究生。
- 风格：干货、经验、踩坑、故事型。

流程：
1. 爆款结构检索：模拟同主题 Top 20 高赞笔记，抽取标题、开头、分段结构和结尾转化。
2. 结构模板压缩：沉淀标题类型、开头钩子、正文结构、行动引导。
3. 内容生成：保留结构，但替换案例、表达、逻辑顺序和关键词。
4. 标题生成：焦虑型、结果型、对比型、数字型、经验总结型共 10 条。
5. 输出 3 个版本：干货版、情绪增强版、转化引导版。
6. 自动入库：飞书多维表格、飞书文档、本地基础数据库。

## 2. 朋友圈内容智能体

输入：
- 场景：带学员开会、收到 offer、复盘。
- 身份标签：导师、学姐、规划师。
- 情绪方向：克制、激励、专业、轻松。

流程：
1. 场景识别：成果型、过程型、思考型。
2. 人设映射：专业感、可信度、克制情绪基调。
3. 结构生成：场景描述、专业动作、学员反馈、方法提炼、克制收尾。
4. 生成 3 种风格：专业理性版、学姐温和版、稍带传播版。
5. 自动优化：删除营销感词汇，增强真实感细节，控制 100-300 字。
6. 入库飞书：记录场景标签、使用版本、转化效果。

## 3. 知识库/内容资产智能体

输入：
- 已生成内容：小红书、朋友圈、招生话术、方法论总结。

流程：
1. 自动分类：小红书笔记、朋友圈内容、招生话术、方法论总结。
2. 结构拆解：标题结构、开头模板、爆款钩子、转化话术。
3. 生成模板库：爆款标题模板、结构模板、朋友圈模板、招生话术库。
4. 标签体系：主题、目的、风格、转化标签。
5. 向量化入库：为后续知识检索和 RAG 提供上下文。
6. 飞书多维表格沉淀：内容 ID、类型、主题、结构模板、标题、正文、转化标签、使用次数、表现反馈。
